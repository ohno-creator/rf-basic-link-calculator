周波数・波長

周波数から波長 λ、λ/2、λ/4、λ/8 を計算します。半波長アンテナの目安や、誘電率による波長短縮も図で確認できます。

はじめての見方

式より先に、ここだけ見れば大丈夫です。

何を決める？

周波数から波長を出し、アンテナ長やアンテナ間隔のサイズ感をつかみます。

何を入れる？

使う周波数を入れます。基板上や材料中で考える場合は誘電率も確認します。

結果をどう読む？

λ/2やλ/4を、アンテナ長、配置間隔、基板配線の最初の目安にします。

周波数・波長計算

周波数から波長 λ、λ/2、λ/4、λ/8 を計算します。

電波が1秒間に振動する回数です。周波数が高いほど波長は短くなります。入力した数値の単位です。MHz=百万Hz、GHz=十億Hz（=1000MHz）。920は「MHz」、2.4や5.8は「GHz」を選びます。単位を切り替えると表示値は自動で換算され、実周波数は保たれます。

波長 λ

電波1周期分の長さ（λ=光速/周波数）です。アンテナ寸法の基準になります。例:920MHzで約33cm。

32.6cm

λ/2

半波長。ダイポールなどが共振する基本アンテナ長の目安です。実際は端部効果で数%短くなります。

16.3cm

λ/4

4分の1波長。モノポール/接地アンテナや整合スタブの基準長です。GND面を半波長の片側代わりに使う構成で用います。

8.1cm

λ/8

8分の1波長。小型化アンテナや整合素子の寸法目安です。短いほど効率・帯域は犠牲になりやすくなります。

4.1cm

アンテナ物理長・短縮率

λ/4・λ/2・5/8λを、端部効果や誘電体・ヘリカル化を想定した短縮率込みで換算します。

理想波長に対する実際の物理長の比率です。外部ホイップなら90〜98%、誘電体や装荷コイルを使う小型アンテナではさらに短くなることがあります。

15.5cm

5/8λ

使い方チュートリアル

  1. 1. まず周波数と単位を合わせます。920MHzならMHz、2.4GHzならGHzが便利です。
  2. 2. λ/4、λ/2、5/8λのどれが検討中のアンテナ形式に近いかを見ます。
  3. 3. 短縮率を動かして、実際の物理長が筐体に入るか確認します。
短縮率
理想的な電気長に対する実際の物理長の割合です。端部効果、誘電体、ヘリカル化で短くなります。
λ/4
モノポールやラジアル、接地型アンテナでよく使う基準長です。
λ/2
ダイポールや半波長ホイップの基準です。スタッフ標準品の920MHz λ/2ホイップの理解にもつながります。
計算式を見る
λ[m] = 299,792,458 / 周波数[Hz]
物理長 = 電気長 × 短縮率

920MHzの場合、波長は約32.6cmです。アンテナ長は単純なλ/4だけでは決まらず、筐体やGND、誘電体の影響を受けます。

周波数が高いほど半波長は短くなり、アンテナも小さくできます。表示値は理想的なλ/2で、実際の共振長は端部効果などで数%短くなります。

なぜ半波長なのか

アンテナが電波を効率よく放射するには、導体の長さが波長と釣り合っている必要があります。長さがちょうど半波長（λ/2）のとき、導体上の電流分布が定在波としてきれいに収まり「共振」します。共振時は給電点インピーダンスが純抵抗（半波長ダイポールで約73Ω）になり、反射が少なく、給電した電力を効率よく電波に変えられます。これが半波長がアンテナ設計の基準寸法になる本質的な理由です。

誘電率による波長短縮と特性への影響（目安）

高い比誘電率εrの材料を使うほどアンテナを小さくできますが、その分だけ帯域や効率は犠牲になります。

空気εr 1基準（短縮なし）×1.00100%ほぼなし
PTFE（テフロン）εr 2.1低損失・ケーブル誘電体×0.6969%小
FR4 基板εr 4.4一般的なプリント基板×0.4848%中
アルミナεr 9.8セラミック基板×0.3232%中
高誘電率セラミックεr 20チップアンテナ材×0.2222%大
高εr セラミックεr 40小型チップアンテナ材×0.1616%特大

表の波長短縮率は λg/λ0 = 1/√εr で、エレメントを誘電体で完全に囲んだ理想（最大）の値です。実際の基板やチップアンテナでは電界の一部しか誘電体内を通らないため、効くのは実効比誘電率 εeff（1〜εr の間）で、短縮量はこれより小さくなります。εeff は、誘電体の厚さ（厚く・十分に囲むほど εr に近づく）と、エレメントの片面だけか両面を覆うか（両面ほど効く）で変わります。帯域の目安は空気（εr 1.0）を100%とした相対値で、寸法縮小に比例させた簡易目安です。実際の帯域・効率は、形状、材料損失（tanδ）、基板GND、実装条件にも左右されます。

半波長を縮める工夫（誘電率・装荷）

製品ではλ/2のままだと大きすぎることが多く、いくつかの小型化手法で物理長を縮めます。

  • 誘電体装荷：比誘電率εrの高い材料の中では波長が λ/√εr に短くなります。セラミックチップアンテナのように、高εr材を使って物理サイズを小さくします。実際の効き方は、誘電体の厚さや、エレメントの片面だけか両面を覆うか（実効比誘電率 εeff）で変わります。
  • メアンダ／ヘリカル形状：導体を蛇行・らせん状に折りたたみ、限られたスペースで電気的な長さを確保します。
  • ローディング（装荷）：途中にコイル（インダクタ）を入れたり、先端に容量を付けたりして、短い導体でも共振させます。
誘電体中の波長 λg = λ0 / √εr
小型化すると特性はどう劣化する？

アンテナを半波長より小さくすると、特性は必ずどこかが犠牲になります。

  • 放射効率の低下：導体や誘電体の損失が相対的に大きくなり、電波に変わらず熱になる割合が増えます。
  • 帯域が狭くなる：Q値が高くなって共振が鋭くなり、使える周波数範囲が狭まります。
  • 利得の低下：放射効率の低下に伴い、利得も下がります。
  • マッチングがシビアに：筐体、基板GND、周辺部品、人体の影響を受けやすく、量産ばらつきにも敏感になります。

これは小型アンテナの物理的な限界（Chu限界）に由来し、「小型化」と「効率・帯域」は本質的にトレードオフの関係です。どこまで小さくして目標性能を満たせるかは、筐体込みの実機評価で見極めるのが確実です。
