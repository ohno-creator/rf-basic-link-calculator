不要ビーム判定（アレイ間隔）

一般的なIoT端末の2本アンテナMIMOというより、ビームを作って方向を振るアレイアンテナ／フェーズドアレイの初期検討に使います。

わかること狙った方向以外にも強いビームが出るかどうかを、間隔と向きから確認します。

アレイアンテナ／フェーズドアレイで、アンテナ素子の間隔が広すぎると現れる不要な強いビーム「グレーティングローブ」が出る条件を確認する計算ツールです。狙った方向のほかに電波の山ができてしまうかを、周波数・素子間隔・走査角から判定します。

何を決める？

複数の素子でビームを作るとき、狙った方向以外にも強いグレーティングローブが出ないかを確認します。

何を入れる？

周波数、素子同士の間隔、ビームを向けたい方向（走査角）を入れます。

結果をどう読む？

不要ビームが出る可能性ありなら、素子間隔を狭めるか、振る角度を小さくします。

狙っていない方向にも強い電波が出るか

アレイアンテナ／フェーズドアレイで素子の間隔を広げすぎると、狙った方向のほかにも「グレーティングローブ」と呼ばれる強い不要ビームが現れます。このページは、それが出る条件を周波数・素子間隔・走査角（向けたい方向）から確認します。

周波数プリセットを起点に、寸法を少しずつ動かしてグラフの傾きを見てください。アンテナ計算では単一の答えより、どの入力が効くかを掴むことが重要です。

計算結果（まずここを見る）

主役の値は大きく表示し、その値で何を判断するかは各カードの補足と下の解説にまとめています。dB、長さ、割合など単位も必ず確認してください。

不要ビーム

狙っていない方向にも強い電波の山が出るかを判定します。

出にくい

式と前提を見る
不要ビームが実際に出る条件: |sinθ0 + mλ/d| ≤ 1
安全目安: d/λ ≤ 1 / (1 + |sinθmax|)

等間隔に並べたアンテナでは、間隔dが広いほど別方向にも同じような山が出やすくなります。式はその山が実空間に現れる条件です。実際の強さはアンテナ単体の形、振幅の付け方、筐体、相互結合で変わりますが、まず危ない配置を見つける一次判定として使えます。

この計算の詳しい解説を読む

狙う方向と余計な方向

入力値から計算した主要指標を、アンテナの形やビーム、距離関係に重ねた概念図です。寸法比は見やすさ優先の模式図です。

この計算で決められること

要するに：複数アンテナで、狙っていない方向に強く飛ばないかを見るツール

アレイアンテナは、複数のアンテナを並べて電波の向きを作ります。ただ、アンテナ同士の間隔が広すぎると、狙った方向とは別の方向にも強い電波の山が出ます。通信なら干渉やムダ打ち、センサーなら誤検知の原因になります。

式を暗記するより、入力を少し動かして結果が急変する場所を見るのが近道です。まず主役の数値、判定、グラフの傾きを見てください。

日常イメージ

懐中電灯を1本だけ向けるなら分かりやすいですが、ライトを等間隔にたくさん並べると、正面以外にも明るい筋が出ることがあります。ここでは、その余計な明るい筋が出るかを見ます。

実際の使われ方

ローカル5G、5G基地局、ミリ波レーダー、MIMOアンテナで、アンテナを何cm間隔で並べるか、どこまで横へビームを向けてよいかを決める初期チェックに使います。

まず触るところ

見るべき結果は「不要ビーム」です。出にくいならまず安心、出る可能性ありならアンテナ間隔を狭めるか、向けたい方向を正面寄りにします。

このページでつまずきやすい語：不要ビームの専門名です。狙った方向以外に出る強い山を指します。ここでは「向けたい方向」のことです。正面から横へどれだけ振るかを角度で表します。波長の半分の間隔です。アレイでよく使われる安全寄りの基準です。

横へ向けるほど、許される間隔は狭くなる

赤い線が今の間隔、青い線が安全寄りの上限です。赤が青を超えると、狙っていない方向にも強く出る可能性があります。

使い方と判断ポイント・用語ミニ辞典

どの順番で入力し、結果を何の判断に使うのかをこのページ内で確認できます。

プリセットを押してから、1つの入力だけを動かしてください。グラフと模式図がどう変わるかを見ると、このツールで何を判断できるかが掴みやすくなります。

3ステップで使う

  1. 1まずプリセット「Sub6 安全寄り」を押します。判定が「出にくい」なら、狙った方向以外に強い山が出にくい条件です。
  2. 2次に「向けたい方向」を大きくします。横へ向けるほど、安全に使えるアンテナ間隔が狭くなることを見ます。
  3. 3「間隔が広すぎる例」を押すと、なぜ問題になるかが分かります。出る可能性ありなら、間隔を狭めるか、向ける角度を小さくします。

このページの用語

不要ビーム
本来向けたい方向とは別に出てしまう強い電波の山です。専門的にはグレーティングローブと呼びます。
アレイ
複数のアンテナを並べ、タイミングをずらして電波の向きを作る仕組みです。
向けたい方向
正面からどれだけ横へ電波を向けるかです。専門的には走査角と呼びます。
λ/2
波長の半分の間隔です。不要ビームを避けるときの代表的な目安です。

コラム：なぜ「間隔を広げればよい」ではないのか

Point 1

アンテナを離すと、アンテナ同士の影響は減り、実装もしやすく見えます。しかし離しすぎると、複数アンテナの周期性が目立ち、狙った方向以外にも強い山が出ます。

Point 2

正面だけを見るならλ/2はよく使われる基準ですが、横へ大きく向けるほど安全な上限はλ/2より小さくなります。

Point 3

このトレードオフをスライダーで動かすと、アレイ設計で『詰めたい理由』と『離したい理由』が同時に見えます。
