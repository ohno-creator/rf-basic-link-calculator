ミスマッチと通信距離

自由空間（距離2倍で−6dB）を前提に、整合ずれの価値を距離％で示す一次目安です。市街地・屋内では距離指数が2から外れ、反射波の再放射やケーブル損による見かけのVSWR変化も含みません。絶対距離の保証ではなく、整合改善の判断材料として使ってください。

わかることVSWRの数字は大きく見えても、失う電力（ML[dB]）は小さく、自由空間で距離に化けるのはさらにその半分です。

VSWR（整合のずれ）からミスマッチ損失ML[dB]と反射電力を求め、自由空間での通信距離への影響を％で概算します。VSWR2でML約0.51dB・距離−5.7%。整合を追い込む価値を、距離の相場観として確認できます。

何を決める？

測定器が示すVSWRが通信距離にどれだけ響くかを％で把握し、整合をどこまで追い込む価値があるかの判断材料にします。

何を入れる？

アンテナアナライザ等が示すVSWRを1つ入れるだけです（スライダー1.0〜10、または1.5／2／3／5のプリセット）。

結果をどう読む？

距離への影響[%]（マイナス表示）とML[dB]・反射電力[%]が出ます。VSWR2で距離−5.7%どまり。MLをリンクバジェット診断の追加損失に足すと、整合ずれ込みの通信余裕を判定できます。

入力条件

測定器が示すVSWR（整合のずれ具合）を1つ入れるだけです。反射で失う電力を ミスマッチ損失[dB]に直し、自由空間ならそれが通信距離にどれだけ響くかを％で見ます。

VSWRプリセット（現場でよく口にする代表点）

読み方: 距離への影響は自由空間（距離2倍で−6dB）を前提にした目安です。 市街地や屋内では距離指数が2から外れるため、絶対距離の保証ではなく 「整合を追い込む価値」を距離％で見るための相場観として使ってください。

通信距離への影響（自由空間）

-5.7%

内訳

ミスマッチ損失 ML0.51 dB
反射電力11.1 %
距離維持率（自由空間）94.3 %
反射係数 Γ0.333

このMLをリンクバジェット診断の追加損失に足すと、整合ずれ込みの通信余裕を判定できます。

基礎から: なぜVSWR2は「距離6%減」で済むのか
Γ = (VSWR−1)/(VSWR+1)
ML[dB] = −10·log10(1 − Γ²)          （反射で門前払いされる電力）
距離維持率 = 10^(−ML/20) = √(1 − Γ²)   （自由空間: 距離∝√電力）
距離影響[%] = (距離維持率 − 1) × 100

① 整合ずれは、電力の「門前払い」です。送信機とアンテナのインピーダンスがずれていると、送った電力の一部がアンテナに 入れずに跳ね返ります。跳ね返る割合が反射係数Γで、VSWRから Γ=(VSWR−1)/(VSWR+1)。VSWR2ならΓ=0.33、反射電力はΓ²＝11%が門前払いされます。水道ホースの継ぎ目で、太さが合わないと 水が一部跳ね返るのに似ています。

② dBに直すと、意外に小さい。門前払いされずにアンテナへ入る電力は1−Γ²＝89%。これをdBにすると ML＝−10log₁₀(0.89)＝わずか0.51dBです。「11%も反射している」の 体感より、失う電力はずっと小さい——これがVSWRの数字に驚きすぎない勘どころです。

③ 自由空間では、ML[dB]の「半分」が距離のdB。自由空間損失は距離の2乗で効く（距離2倍で−6dB＝20log₁₀）ので、電力のdBを距離のdBに 換算するときは20で割ります。よって距離維持率＝10^(−ML/20)＝√(1−Γ²)。 VSWR2なら√0.89＝0.943、つまり距離は約6%減るだけ。 「VSWR2は距離6%減」が現場の相場観です。 ※このホースのたとえは直感用で、実際の反射波は戻ってまた再反射し、線路上に定在波 （立つ波）をつくります。単に「跳ね返して捨てる」より複雑で、だからこそVSWR＝定在波比という名前がついています。

結論: VSWRの数字は大きく見えても、失う電力（ML[dB]）は小さく、距離に化けるのは さらにその半分。整合を1.05まで追い込むより、アンテナの位置を数十cm動かす方が 距離に効く場面が多いのは、この数字の非対称性が理由です。

コラム：VSWR 2.0で上司に怒られたら読む話

現場でアンテナアナライザを当てると、上司が画面をのぞき込んで言います。「VSWR 2.0？ ダメだろ、1.5切るまで追い込め」。よくある問答です。でも、その2.0が距離に与える影響を 計算すると——失う電力は0.51dB、自由空間での距離減はわずか5.7%。 10km届く設計なら約9.4kmまで、という程度の話です。数字の見た目（2倍！）と、実際に 失うもの（電力の1割・距離の数％）のあいだには、大きなギャップがあります。

VSWRが怖いのは、実は距離が縮むことよりも別の理由からです。反射波が送信機に戻ると 終段が熱を持ち、電力が大きい局では素子を痛めます。だからメーカは「VSWR 2.0以下で 使え」と保護のために規定する。つまりVSWR規格の多くは飛距離の規格ではなく、 機器保護の規格なのです。これを混同すると、距離のために整合を追い込んでいるつもりが、 実は保護マージンを削っているだけ、ということになりかねません。

VSWR 1.05を追い込む職人がいます。無響室で、ケーブルの取り回し一本まで整えて、 小数第二位を削る。その技術は本物です。ただ屋外に出た瞬間、アンテナが雨に濡れれば 誘電率が変わってVSWRは1.8に跳ね上がり、雪が積もればさらに動く。実運用のVSWRは環境で秒単位に呼吸している——1.05という値は、 その一瞬の静止画にすぎません。

では何が本当に距離を決めるのか。順番はこうです。まずアンテナの置き場所と高さ（フレネルゾーンが抜けるか、地面反射を拾わないか）。次に偏波の向き。 その後にようやく整合の追い込みが来ます。VSWRを2.0から1.2に改善して稼げるのは距離4%弱。 同じ努力でアンテナを数十cm高くできるなら、そちらの方がはるかに効く場面が多い。 測定器の数字が一番目立つからといって、それが一番効くとは限らない——現場のいちばん 大事な感覚です。

深掘り（式と出典）

反射係数 Γ = (VSWR−1)/(VSWR+1)。反射電力率 = Γ²、 アンテナへ入る電力率 = 1−Γ²。ミスマッチ損失 ML[dB] = −10·log₁₀(1 − Γ²)。 リターンロスとは RL[dB] = −20·log₁₀(Γ) の関係で、 VSWR2 は Γ=1/3・RL≈9.54dB・反射電力11.1%・ML≈0.51dB。

自由空間では受信電力が距離の2乗に反比例する（FSPL ∝ d²、距離2倍で−6dB＝20log₁₀）ため、 電力のdBを距離のdBへ換算する係数は20。よって距離維持率 = 10^(−ML/20) = √(1 − Γ²)。VSWR2 で √0.889 ≈ 0.943、距離影響 −5.7%。実環境では距離指数が2から外れる（市街地で3〜4）ため、 距離%はあくまで整合改善の価値を測る一次目安である。

なお ML は「反射で失う分」のみを表し、線路損があると反射波が往復で減衰して見かけのVSWRが 良く見えるなど、実測解釈には注意を要する。

出典: D. M. Pozar, Microwave Engineering, 4th ed., Wiley（Ch.2「反射係数・ VSWR・ミスマッチ損失」）／整合とリターンロスの定義は同書 §2.3 による。
