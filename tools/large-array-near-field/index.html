はじめての見方

式より先に、ここだけ見れば大丈夫です。

何を決める？

大型アレイや高周波アンテナを、遠方界として扱ってよい距離かを確認します。

何を入れる？

結果をどう読む？

大型アレイ近傍界・遠方界判定

大型アレイや高周波アンテナで、相手や測定点を「十分遠い」とみなしてよいかを確認します。測定距離、ビーム計算、近傍界補正が必要かを判断するための計算です。

周波数プリセットを起点に、寸法を少しずつ動かしてグラフの傾きを見てください。アンテナ計算では単一の答えより、どの入力が効くかを掴むことが重要です。

この計算で決められること

要するに：大型アレイを、遠方界として扱ってよい距離かを見るツール

アンテナの開口が大きくなると、遠方界になる距離が想像以上に長くなります。近い相手には、角度だけでなく距離方向の焦点も効いてきます。

式を暗記するより、入力を少し動かして結果が急変する場所を見るのが近道です。まず主役の数値、判定、グラフの傾きを見てください。

日常イメージ

遠くの山は平らな背景のように見えますが、目の前の大きな看板は端と中央で見える角度が違います。大型アレイでも似た見え方の差が出ます。

実際の使われ方

Sub6の大きな反射板、ミリ波アレイ、6G/XL-MIMO、アンテナ測定距離の検討で、遠方界前提でよいかを確認します。

まず触るところ

一番大きい幅を2倍にしてください。遠方界になる目安距離が4倍近く伸びることを見ると、大型アレイの測定距離が問題になる理由が分かります。

このページでつまずきやすい語：アンテナに近く、波の曲がりや距離方向の違いを無視しにくい領域です。十分遠く、波を平面波として扱いやすい領域です。遠方界とみなす代表的な目安距離です。開口サイズの2乗に比例します。

計算結果（まずここを見る）

主役の値は大きく表示し、その値で何を判断するかは各カードの補足と下の解説にまとめています。dB、長さ、割合など単位も必ず確認してください。

遠方界になる目安

この距離より十分遠いと、平面波・遠方界として扱いやすくなります。

46.70m

近さの指標

D²/(λR)。1以上なら近傍界性が強く、距離方向の違いを無視しにくい目安です。

2.33

遠方界扱い

評価距離が遠方界の目安より近いかどうかを見ます。

要注意

この距離では遠方界前提に注意が必要です。近傍界ビーム、測定距離、位相補正、距離方向の焦点を確認してください。

式と前提を見る
Rff = 2D² / λ
Fresnel数 = D² / (λR)
端部の経路差 = √(R²+(D/2)²)-R

大型アレイや高周波アンテナでは、相手が遠方界にいるとは限りません。この計算は、測定距離や通信距離で遠方界前提を使ってよいかを確認するためのものです。近傍界では平面波ではなく球面波として扱う必要があり、ビームは角度だけでなく距離にも焦点を持ちます。

この計算の詳しい解説を読む

入力値から計算した主要指標を、アンテナの形やビーム、距離関係に重ねた概念図です。寸法比は見やすさ優先の模式図です。

アンテナ面が大きいほど、必要距離は急に伸びる

遠方界距離はD²/λで増えるため、開口を大きくすると急に伸びます。測定環境や解析モデルの確認に使います。

使い方と判断ポイント・用語ミニ辞典

どの順番で入力し、結果を何の判断に使うのかをこのページ内で確認できます。

プリセットを押してから、1つの入力だけを動かしてください。グラフと模式図がどう変わるかを見ると、このツールで何を判断できるかが掴みやすくなります。

3ステップで使う

  1. 1まず、アンテナ面の最大寸法と、相手や測定点までの距離を入れます。結果の「遠方界になる目安」を見ます。
  2. 2相手までの距離がその目安より短い場合、遠方界前提の利得測定やビーム計算では外す可能性があります。
  3. 3最後に開口サイズを2倍にしてみます。遠方界距離が急に伸びるため、大型アレイほど測定距離が問題になることが分かります。

このページの用語

Fraunhofer距離
遠方界とみなす代表的な距離目安です。2D²/λで計算します。
Fresnel数
D²/(λR)で表す近傍界性の目安です。大きいほど球面波性が強くなります。
近傍界
波面の曲率や距離方向の焦点が無視できない領域です。
XL-MIMO
非常に大きなアンテナ数/開口を持つMIMOです。6G研究で近傍界性が注目されています。

コラム：大型アレイでは『十分遠い』が意外と遠い

Point 1

6G/XL-MIMOではアンテナ数や開口が大きくなり、従来の平面波・遠方界モデルだけでは足りない場面が増えています。

Point 2

近傍界ではビームステアリングだけでなく、ビームフォーカシング、球面波、空間非定常性が問題になります。

Point 3

このツールは発展的な計算の入口として、まず2D²/λがどれほど大きくなるかを体感するためのものです。
