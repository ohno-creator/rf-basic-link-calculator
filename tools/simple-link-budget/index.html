かんたんリンク計算

送信電力、アンテナ利得、自由空間損失、追加損失、受信感度だけで、受信電力とリンクマージンを計算します。RFリンクバジェットの足し引きを最小構成で確認できます。

対象：リンクバジェットの考え方を最短で確認する入口です。壁・地面反射・筐体・実測補正まで分けたい場合は、総合版のリンクバジェット診断を使います。

はじめての見方

式より先に、ここだけ見れば大丈夫です。

何を決める？

通信が届きそうかを、受信電力と受信感度の差だけでまず確認します。

何を入れる？

周波数、距離、送信電力、受信感度を入れます。分かる場合だけアンテナ利得や追加損失を足します。

結果をどう読む？

リンク余裕が0dBを下回ると厳しく、10〜20dB以上あるほど設置ばらつきに強くなります。

用語が分からない場合は、ページ内の「？」や用語ミニ辞典を開いてください。まずは近いプリセットや初期値から動かすと、変化がつかみやすくなります。

かんたんリンク計算

送信電力から距離損失と追加損失を引き、受信感度に対して何dB余裕があるかだけを見ます。

通信距離送信側と受信側の距離です。屋内や机上の確認は m、屋外の見通し確認は km が扱いやすいです。

必要なら少しだけ補正

まずは0のままで構いません。アンテナ利得が分かる、壁や筐体の追加損失を仮置きしたい、という時だけ入れます。

結果

自由空間損失だけで見る、最小構成のリンク余裕です。

0dBラインとの距離

リンク余裕0dBは、受信電力と受信感度がちょうど同じ位置です。右に行くほど余裕があり、左側は不足です。表示範囲は -30〜+30dB です。
-30dB0dB+30dB

十分なリンクマージンがあります。通常の設置ばらつきや軽微な環境変動があっても、通信が成立しやすい条件です。

足し引きの中身

dBmの電力に、dB/dBiの利得と損失を足し引きして受信電力を求めます。最後に受信感度を引いたものがリンク余裕です。
送信電力13.0 dBm
+アンテナ利得（合計）0.0 dBi
-自由空間損失91.7 dB
-追加損失10.0 dB
=受信電力-88.7 dBm
-受信感度-120.0 dBm
=リンク余裕+31.3 dB
一番小さなリンクバジェット
受信電力[dBm] = 送信電力[dBm] + アンテナ利得[dBi] - FSPL[dB] - 追加損失[dB]
リンク余裕[dB] = 受信電力[dBm] - 受信感度[dBm]

この簡易版は、伝搬損失を自由空間損失だけで見ます。壁、地面反射、筐体、人体、設置方向の影響を分けたい場合は、総合版のリンクバジェット診断で確認してください。
