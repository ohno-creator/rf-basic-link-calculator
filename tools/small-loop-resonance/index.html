小型ループアンテナ共振

NFCやRFIDなどの小さなループを狙った周波数に合わせるため、ループ形状からインダクタンスと必要な同調コンデンサ容量を概算します。

はじめての見方

式より先に、ここだけ見れば大丈夫です。

何を決める？

小さなループを狙った周波数に合わせるため、載せるコンデンサ容量の目安を出します。

何を入れる？

狙う周波数、ループ直径、導体の太さ、巻数を入れます。

結果をどう読む？

必要容量が極端なら、コンデンサだけでなくループ径や巻数を見直します。

用語が分からない場合は、ページ内の「？」や用語ミニ辞典を開いてください。まずは近いプリセットや初期値から動かすと、変化がつかみやすくなります。

小型ループアンテナ共振

小さな輪っか状アンテナを狙った周波数に合わせるため、どのくらいのコンデンサを入れればよいかを見ます。NFC、RFID、近距離センサーの部品選定の入口です。

周波数プリセットを起点に、寸法を少しずつ動かしてグラフの傾きを見てください。アンテナ計算では単一の答えより、どの入力が効くかを掴むことが重要です。

この計算で決められること

要するに：小さな輪っかに、どのコンデンサを載せれば狙いの周波数になるかを見るツール

小型ループは、そのままでは狙いの周波数に合わないことが多いので、コンデンサを足して共振させます。このページは、ループのLと必要なCの目安を出します。

式を暗記するより、入力を少し動かして結果が急変する場所を見るのが近道です。まず主役の数値、判定、グラフの傾きを見てください。

日常イメージ

ギターの弦を張り具合でチューニングするように、ループも容量で『その周波数に合わせる』と考えると入りやすいです。

実際の使われ方

NFC、RFID、近距離センサー、低周波の小型アンテナで、試作前に必要な容量レンジや、巻数を増やした時の変化を確認します。

まず触るところ

NFC 13.56MHzプリセットを押し、ループ直径と巻数を動かしてください。必要容量が大きく変わることを見るのがポイントです。

このページでつまずきやすい語：インダクタンスです。ループが磁界としてエネルギーをためる力です。容量です。Lと組み合わせると特定の周波数で共振します。アンテナや回路が特定の周波数で反応しやすくなる状態です。

計算結果（まずここを見る）

主役の値は大きく表示し、その値で何を判断するかは各カードの補足と下の解説にまとめています。dB、長さ、割合など単位も必ず確認してください。

ループのL

ループが磁界としてエネルギーをためる量です。ここから必要容量を決めます。

94.7nH

載せる容量の目安

指定周波数でループを合わせるためのコンデンサ容量の初期候補です。

1454.6pF

小型近似の確認

ループ周囲長が波長に対して十分小さいかを見る目安です。

0.006λ

式と前提を見る
L ≈ μ0N²r(ln(8r/a)-2)
C = 1 / ((2πf)²L)

単純な円形導体ループの近似です。この計算は、狙う周波数に合わせるためのコンデンサ容量レンジを決める入口です。プリントループや多巻きコイルでは寄生容量、近接効果、基板、手や筐体の影響が大きいため、初期値として扱ってください。

この計算の詳しい解説を読む

ループに足すコンデンサを決める

入力値から計算した主要指標を、アンテナの形やビーム、距離関係に重ねた概念図です。寸法比は見やすさ優先の模式図です。

狙う周波数で、必要容量がどう変わるか

同じループなら、周波数が高いほど必要な同調容量は小さくなります。部品値の候補が現実的かを見ます。

使い方と判断ポイント・用語ミニ辞典

どの順番で入力し、結果を何の判断に使うのかをこのページ内で確認できます。

プリセットを押してから、1つの入力だけを動かしてください。グラフと模式図がどう変わるかを見ると、このツールで何を判断できるかが掴みやすくなります。

3ステップで使う

  1. 1まず、狙う周波数とループの大きさを入れます。結果の「必要な共振容量」が、部品選定の最初の候補です。
  2. 2巻数や線径を動かし、必要容量が手に入りやすい範囲に入るかを見ます。極端な容量ならループ形状を見直します。
  3. 3最後に周長/λを見ます。値が大きくなると、小型ループとしての単純近似から外れやすくなります。

このページの用語

インダクタンス L
ループが磁界としてエネルギーを蓄える度合いです。巻数の2乗で増えやすい量です。
共振容量 C
指定周波数でLC共振させるために必要な容量です。寄生容量も含めて考えます。
周長/λ
ループ周囲長が波長に対してどれだけ大きいかの比です。小型近似の目安になります。
Q
共振の鋭さです。高いほど選択度は高い一方、帯域は狭くなります。

コラム：部品値が極端なら、形状から見直す

Point 1

必要容量が極端に小さい、または大きい場合は、コンデンサ選定だけでなくループ径や巻数を見直した方が早いことがあります。

Point 2

小型ループは磁界結合や近距離用途で便利ですが、寄生容量や手・筐体の影響を受けやすい構造です。

Point 3

高いQは選択度を上げる一方、帯域や量産ばらつきの許容を狭くします。試作では可変容量や複数値で追い込みます。
