## 2. 定量値の引用 (Quantitative Data)

1. **可視空間内（$\pm 90^\circ$）でグレーティングローブを完全に回避するアレイ素子間隔の限界式**
   - **数値**: 
     $$d_{\text{max}} < \frac{\lambda}{1 + |\sin \theta_0|}$$
     ここで $d$ は隣接素子間隔、$\lambda$ は波長、$\theta_0$ はビーム走査角（正面を0度とする）。
   - **代表的なしきい値**:
     - **ビーム走査をしない（正面固定照射 $\theta_0 = 0^\circ$）場合**: **$d_{\text{max}} < \lambda$** (1波長以上でグレーティングローブが出現)。
     - **全角走査（最大 $\theta_{0,\text{max}} \to \pm 90^\circ$）を行う場合**: **$d_{\text{max}} < 0.5 \lambda$** (半波長以上でグレーティングローブが出現)。
     - **走査角を最大 $\theta_0 = 45^\circ$ に限定する場合**: **$d_{\text{max}} < 0.586 \lambda$**。
     - **走査角を最大 $\theta_0 = 60^\circ$ に限定する場合**: **$d_{\text{max}} < 0.535 \lambda$**。
   - **出典**: [S1] Balanis Chapter 6.3.3 / [S2] Mailloux Chapter 3。

2. **グレーティングローブの発生限界比熱減衰**
   - **数値**: グレーティングローブが出現すると、それはメインビームと **全く同じピーク強度（0 dB差）** で不要な方向に放射されます。これにより、本来向けたかった方向へのメインビーム電力は **半分以下（-3 dB以上）** に分散・損失します。
   - **出典**: [S2] Mailloux Chapter 3。

## 3. 導出メモ (Derivation Notes)

1. **グレーティングローブの物理的発生メカニズム (空間エイリアシング)**
   - アレイアンテナは、個々のアンテナ素子からの電波の位相差を制御して合成することでビームを特定の方向に向けます。
   - これは、デジタル信号処理における「サンプリング定理」に非常に似ています。空間的なサンプリング間隔である「素子間隔 $d$」が粗すぎると（波長に対して広すぎると）、空間的な折り返し雑音（エイリアシング）が発生し、別の角度でも各素子からの電波の位相が完全に揃う（コヒーレントに強め合う）極大点が出現します。これがグレーティングローブです。

2. **可視領域（Visible Space）の境界条件**
   - 電波が伝搬できる実空間は、角度 $\theta$ にして $-90^\circ$ から $+90^\circ$（すなわち $|\sin\theta| \le 1$）の領域です。これを「可視領域」と呼びます。
   - 素子間隔を広げると、可視領域の外にあった無効電力成分（リアクティブフィールド内のグレーティングローブ）が、ビーム走査角 $\theta_0$ を傾けることによって実空間（可視領域内）に「引っ張り込まれて」伝搬波に変化し、不要ビームとなって飛び出します。

## 4. アンチパターン (Anti-Patterns)

1. **【誤解】「アレイアンテナのビーム幅をシャープにするため、素子数を変えずに、実装スペースが余っているからと素子間隔を 1λ 近くに広げて並べる」**
   - **数値の帰結**: ビームを正面から少し横（例: 走査角 30度）へ向けた瞬間、メインビームと全く同じ強度の強烈な不要ビーム（グレーティングローブ）が反対側の側方（例: -30度付近など）に放射され、他システムへの深刻な電波干渉、あるいはレーダー・センサーモジュールでの致命的な **方向探知の誤検知（像の二重映り）** を引き起こします。
   - **対策**: アレイ設計では、ビーム幅のシャープさは「間隔の拡大」ではなく「素子数の追加（アレイ全体の開口径拡大）」によって実現し、個々の素子間隔 $d$ は必ず走査角に応じた $d < \frac{\lambda}{1 + |\sin\theta_{max}|}$ の物理限界（原則 $0.5 \lambda$）の内に抑えます。
