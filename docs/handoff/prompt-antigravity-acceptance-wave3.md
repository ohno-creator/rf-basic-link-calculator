## タスク① リンクバジェット「かんたんモード」の官能評価（最優先）
`/tools/rf-basic-link-calculator/`（既定=かんたんモード）について、**「入力を迷わないか」「直感的に操作できるか」**を初心者視点で実機評価:
1. 初見でSTEP1→2→3の流れに迷いがないか（各カード文言の分かりにくい箇所を具体指摘）
2. 「次の一手」チップ: 表示内容の妥当性・クリック適用後の結果変化が理解できるか・逆算距離の納得感
3. 距離対数スライダーの操作感（1m〜20km・m/km表示切替の自然さ）
4. モバイル(375px)での操作性: 固定結果バー・タッチ44px・横はみ出しなし
5. かんたん⇔詳細の切替: 値の引き継ぎ・詳細側のクイック調整/解説付き切替との関係が混乱しないか
6. 共有URL(?付き)で開くと詳細モードで開く挙動の確認
