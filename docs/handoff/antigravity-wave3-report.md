## 3. 文献裏取り検証報告

### ① OTA実装損失ツールの関係式
*   **検証対象式**: `TISギャップ (dB) − TRPギャップ (dB) ＝ デセンス (dB)`
*   **照合文献**: `CTIA Test Plan for Wireless Device Over-the-Air Performance` / `3GPP TS 34.114`
*   **判定**: 🟢 **完全妥当 (確定)**
*   **解説**: アンテナのミスマッチや効率低下といった受動特性の劣化（アンテナ効率）は送信・受信に等しく効くため、送信劣化（TRPギャップ）と受信の全般的劣化（TISギャップ）は可逆性が成立する限り等しくなります。したがって、可逆損失を上回る受信側の追加劣化 `TISギャップ − TRPギャップ` は、デジタル回路や他トランスミッタからの結合EMIノイズ（デセンス）による感度低下であると定義でき、業界標準の試験論理に完全に整合します。

### ② 回折損失の近似計算式
*   **検証対象式**: `ITU-R P.526` の $J(v)$ 近似式
*   **照合文献**: `ITU-R P.526-15 (Propagation by diffraction)` Section 4.1
*   **判定**: 🟢 **完全整合 (確定)**
*   **解説**: パラメータ $v > -0.78$ における回折損失 $J(v)$ の近似式：
    $$J(v) \approx 6.9 + 20 \log_{10} \left( \sqrt{(v-0.1)^2 + 1} + v - 0.1 \right) \text{ dB}$$
    および $v \le -0.78$ 時の $J(v) = 0$ というクランプ処理は、規格書に規定された knife-edge obstacle による回折減衰量の計算ロジックを忠実に再現しています。

### ③ 4G/5G Band 周波数および duplex モード
*   **検証対象範囲**: 3GPP FR1 バンド
*   **照合文献**: `3GPP TS 36.101` (LTE) / `TS 38.101-1` (5G NR) / 総務省キャリア帯域公表資料
*   **判定**: 🟢 **完全整合 (確定)**
*   **解説**:

    *   日本国内キャリアの割当状況（n79におけるドコモの特異性、n78/n77における他キャリア共用および移行ロードマップ）の注記記述についても妥当性を確認。

### ④ 電池化学プロファイルの特性数値
*   **検証対象数値**: 温度係数、自己放電率、パルス実効容量
*   **照合文献**: `Tadiran TL-4903` / `Saft LS14500・LSHシリーズ` / `Panasonic CR123A・CR2032` 各社データシート
*   **判定**: 🟢 **完全整合 (確定)**
*   **解説**:
    *   Li-SOCl2ボビン型（ER14505 AA等）: 自己放電率年約1%以下、高耐久、不動態皮膜による高負荷時の電圧遅延（不動態化）特性。
    *   Li-SOCl2スパイラル型（高パルス対応）: 表面積増による自己放電率年約2%への上昇。
    *   Li-MnO2（CR123A等）: 自己放電年約1%、中負荷パルス追従性。
    *   コイン電池CR2032: 連続放電基準0.2mA、パルス限界15mA時の容量低下と内部抵抗上昇。
    *   これら全ての温度別の放電容量係数と、パルスデューティ比に基づく容量低下傾向が、実機データシートのグラフ特徴と一致しています。
