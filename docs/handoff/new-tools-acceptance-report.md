## 2. タスク①: 新ツール13本実機受入検証結果

全13ツールにおいて、以下の共通チェック項目をデスクトップ（1280px）およびモバイル（390px）の両環境で実機検証しました。

### 共通チェック項目
1. **即時応答**: 主結果（`primary-result`）が初期状態で表示され、入力欄への入力やプリセットチップクリックで即座に応答し再計算されること。
2. **動的SVG図**: 動的SVG図が入力値に連動して変形/伸縮すること。また、「図を保存 (SVG/PNG)」ボタンから正常にダウンロードでき、文字化けやサイズ・桁ズレがないこと。
3. **解説・コラム**: 「わかること」ピル、数式解説カード（`FormulaExplanationCard`）、および折り畳み式詳細コラムが崩れなく表示されること。
4. **操作性 & アクセシビリティ**: スライダーやチップ等のタッチ領域が 44px 以上確保されていること、キーボードフォーカスリングが可視であること、`prefers-reduced-motion` でアニメーションが停止すること。
5. **数値の妥当性**: E2Eテストの期待値と一致する数値を手入力し、正しい演算結果が得られること。

## 3. タスク②: 文献パック裏取り検証結果

[track-g-data-literature-pack.md](file:///Users/pc141/Documents/RF%20Basic%20Link%20Calculator/docs/handoff/track-g-data-literature-pack.md) に定義されている G2 / G3 / G4 / G5 / G14 / G15 / G18 各ツールの定数・しきい値テーブルについて、一次出典との突合およびURL生存確認を実施しました。

1. **G2〜G15 項目**:
   * すべて一次出典（TI AN058, Ignionクリアランス仕様, Antenova白書, 3GPP TR, ITU-R P.2040, NISTIR 6055）と一致しており、テーブルしきい値が設計基準として極めて妥当であることを検証し、`✅確認済` を追記しました。
2. **G18 (LoRa ToA & ARIB STD-T108) 項目**:
   * ARIB STD-T108 の特定小電力無線規制（キャリアセンス、送信時間、休止時間）は適合。
   * ToA（エアタイム）計算において、一部プロトコル依存 of オーバーヘッドを含んだ数値と、純粋な Raw LoRa 理論式との間に軽微な乖離を発見したため、**Raw LoRa 理論式に準拠した「正確な ToA テーブル」への差し替え修正提案**をレポート内に追記しました。確定判定は次段の Claude レビューに引き継ぎます。
3. **コラム文献パック (D-b 4本)**:
   * コラム文献パック追加の過程で、[column-small-loop.md](file:///Users/pc141/Documents/RF%20Basic%20Link%20Calculator/docs/handoff/column-small-loop.md) に記載されていた E. B. Rosa (1908) の NIST 用 PDF リンクが 404（リンク切れ）になっていることを発見したため、安定したDOIリンク（`https://doi.org/10.6028/bulletin.088`）へ差し替える修正を施しました。
   * また、[column-radiation-resistance.md](file:///Users/pc141/Documents/RF%20Basic%20Link%20Calculator/docs/handoff/column-radiation-resistance.md) の短縮モノポール放射抵抗公式において、変数 $h$ に対する係数が $10$ になっていた不整合を、鏡像法（モノポールは全長 $l=2h$ のダイポールの半分）に基づく物理的に正しい係数 $40$ に修正し、注意書きを添えました。
