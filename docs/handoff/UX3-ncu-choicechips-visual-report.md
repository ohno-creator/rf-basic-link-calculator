## 2. 視覚検証結果

### 2.1 描画崩れ・レイアウト影響の有無
- **結果**: 良好。
- **詳細**:
  蓋・地表面、BOX・ピット材質、水分・湿潤状態、アンテナ位置、開口・隙間、地表上の遮蔽 の 6 箇所について、従来の個別実装から共有 `@/components/ChoiceChips` へ完璧に移行されています。
  チップ個々の高さ（py-1.5 → py-2）が UI Kit v2 §2.5 基準（40px以上）に統一されたことで、余白設計が綺麗に整い、操作性が著しく向上しました。
  `withChipSeverity` ヘルパーにより、減衰特性（危険度）に応じたチップのトーン変化（赤・黄等のカラーリング）も破綻なく正確に動作していることを目視確認済みです。
- **他要素・他ツールへの影響**: なし。滝グラフや 2D断面図も正常に表示されています。

### 2.2 スナップショット再ベースライン
- Linux/CI 上で Playwright が生成した最新の actual 画像に基づき、基準スナップショット画像（`*-chromium-linux.png`）を上書き更新してコミット・プッシュを完了いたしました。本 PR の機能ゲートテストは完全にグリーン（PASS）となっており、即マージ可能です。
